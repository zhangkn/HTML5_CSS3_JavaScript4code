Hello World -- 第一个CSS3+DIV页面
实例
以下代码是我的第一个CSS3+DIV页面，该页面红色居中显示字符串 Hello World!。 
<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
        <title>Hello World</title>
        <style type="text/css">
        h1 {
            color: #FF0000;
            text-align: center;
        }
        </style>
    </head>
    <body>
        <h1>Hello World! </h1>
    </body>
</html>
